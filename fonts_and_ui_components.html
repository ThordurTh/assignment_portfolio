Assignment 1: Working with Fonts and UI Components

Implementing font hierarchy, UI card design and colours to a mockup landing page design.

Website

Description

This assignment was split into two parts, fonts and then UI cards. Implementing a font hierarchy is essential for a good balance between font sizes. It also makes it easier while designing to establish certain layout rules such as margins, padding and line height, before putting sections together. Knowing how to pair fonts together is also good to know since they have to compliment each other instead of fighting for the viewers attention.

Process

Fonts

I started out by finding a good font hierarchy using type-scale. I picked out a major-triad scale which multiplies 1.25 to the base size (16px) for each heading.

Next step was to find a font pair that I found match the website. I decided to go with a font pair that was recommended by figma. They are both sans-serif but work well together. They are quiet similar although Raleway is a bit slimmer with slightly more details which suit headings. Cabin on the other hand is a bit bolder and simpler which makes for easy reading.

Colour Palette

UI Cards

Next step was to improve the card design and add colours from a colour palette. After doing research on common card trends did I come to the conclusion on 3 things. Borders are rarely used nowadays, if corners are rounded then they are only slightly rounded, and lastly some indication that tells the user that this is a card-like component which either includes a blurred shadow to give it depth or in my case a non-blurred shadow.
