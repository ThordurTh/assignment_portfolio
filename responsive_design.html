Assignment 3: Responsive Design

Applying CSS Flex and Grid methods to a mockup landing page to make it responsive.

Website

Description

This assignment had a focus on applying CSS Flex and Grid methods to make the website suited for devices from mobile phones to desktop computers. I also used the opportunity to apply what I had previously learned to this assignment such as font hierarchy and a colour pallete.

Process

Colour Palette

All images used are from websites such as Unsplash and I made the colour palette based on what I found. I then proceeded to make colour contrast checks and other colour combinations testing before implementing it to the design.

Grid and Flex

I made use of grid and flex to the best of my knowledge since we live in times where these methods are available in CSS.

Responsive Design

There are many methods that are useful for making sure a website is responsive such ase using responsive units on images, sections and fonts but I also made use of media queries.
